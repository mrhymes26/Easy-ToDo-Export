# Release Notes - Version 1.0.1

## Release Date
January 2025

## What's New

### Improvements
- **Enhanced Authentication**: Improved error handling for authentication failures
- **Better Token Management**: More reliable token caching and refresh mechanism
- **UI Improvements**: Updated localization strings for better user experience

### Bug Fixes
- Fixed token refresh issues in `GetAccessTokenAsync` method
- Improved error messages for clearer user feedback
- Better handling of expired tokens

## Installation

1. Download `TodoExport-v1.0.1-win-x64.zip`
2. Extract all files to a folder
3. Run `TodoExport.exe`
4. Sign in with your Microsoft account

## System Requirements

- Windows 10 or Windows 11
- .NET 8.0 Runtime (included in the release)
- Microsoft account with Microsoft To Do

## Upgrade from 1.0.0

If you're upgrading from version 1.0.0:
- Your existing token cache will continue to work
- No data migration required
- Simply replace the old files with the new ones

None at this time.
