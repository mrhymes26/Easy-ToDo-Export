# LinkedIn Project Description

## Microsoft To Do Export Enterprise

Designed and architected a Windows desktop solution that integrates with Microsoft Graph API to enable seamless export of Microsoft To Do data to multiple formats (Excel, CSV, JSON, Text). The Enterprise Edition is specifically configured for Work and School accounts (organizations) using Azure AD organizational authority. The solution implements secure authentication using Microsoft's Device Code Flow and provides a user-friendly interface for data backup and migration scenarios in enterprise environments.

**Skills:** Solution Architecture | Microsoft Graph API | Microsoft Identity Platform (MSAL) | Azure AD | Enterprise Authentication | Cloud Integration | Data Export Solutions | .NET Framework
